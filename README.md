# Kantouzin's dotfiles

## Requirements

[thoughtbot/rcm](https://github.com/thoughtbot/rcm)

## Install

```Shell
git clone https://github.com/Kantouzin/dotfiles.git $HOME/dotfiles
env RCRC=$HOME/dotfiles/rcrc rcup
```

## Register dotfile

```Shell
mkrc .dotfile
```

## Update

```Shell
rcup
```
